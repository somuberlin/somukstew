Portfolio Details

This Animation Website UI Design was created using Figma, focusing on a clean and engaging user experience. The design features a modern layout with smooth transitions, micro-interactions, and intuitive navigation that enhances user engagement. I incorporated bold typography, a balanced color palette, and animated UI elements to bring the interface to life. The design is fully responsive and optimized for both desktop and mobile viewing. This project highlights my skills in UI/UX design, animation planning, and creative prototyping, with special attention to user flow and visual hierarchy.
